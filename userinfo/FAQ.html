FAQ

  • 결제 진행이 원할하지 않은 경우

    인터넷 브라우저를 바꿔서 시도해보시거나, 브라우저를 초기화를 진행해주십시오.


    크롬 초기화

    • 설정 > '기본값' 검색 > '설정을 기본값으로 복원' 클릭

    • 설정 초기화 화면에서 '설정 초기화' 버튼 클릭

  • 온라인 신청시 승인확인 메세지가 뜨는 경우

    인터넷 익스플로러 사용 시

    1. 우측 상단의 설정(톱니바퀴) 클릭

    2. "호환성 보기 설정"

    3. 리스트에 "katri.re.kr" 추가

    4. 인터넷 전부 끄고 다시 시도합니다.
