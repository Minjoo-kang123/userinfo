회원정보 FAQ

  • 아이디 찾는 방법

    아이디가 기억나지 않는다면, 로그인 화면 하단의 아이디 찾기를 통해 아이디를 확인할 수 있습니다.


    아이디 찾기

    • 기업 회원의 경우 기업회원 > 기업명, 사업자등록번호 기입 후 확인 클릭

    • 개인 회원의 경우 개인회원 > 이름, 이메일 기입 후 확인 클릭

    이후 가입시 메일로 회신된 ID 확인

  • 비밀번호 찾는 방법

    비밀번호가 기억나지 않는다면, 로그인 화면 하단의 비밀번호 찾기를 통해 비밀번호를 확인할 수 있습니다.


    비밀번호 찾기

    • 기업 회원의 경우 기업회원 > 아이디, 기업명, 사업자등록번호 기입 후 확인 클릭

    • 개인 회원의 경우 개인회원 > 아이디, 이름, 이메일 기입 후 확인 클릭

    이후 가입시 메일로 회신된 임시 비밀번호 확인

  • 성적서 조회가 불가한 경우

    현 계정이 기업회원인지 개인회원이지 확인해주시길 바랍니다.

    기업에서 요청한 성적서의 경우 개인회원 아이디로 조회가 불가합니다.

    따라서 해당 아이디 확인 후 다시 조회하시길 바랍니다.

  • 회사명을 변경하고 싶어요.

    회사명 변경은 회원 탈퇴 후 재가입을 통해 변경이 가능합니다.

    사업자 번호가 변경되지 않은 경우, 회사명을 변경하지 않고 사용이 가능합니다.

    사업자 번호와 회사명 두가지 모두가 변경 된 경우에는

    탈퇴 후 재가입으로 수정하셔야 합니다. 

  • 아이디/비밀번호 찾기 메일이 안와요

    아이디 비밀번호 찾기 후 메일이 오지 않는 경우 담당자의 메일이 변경되지 않았는지 확인해주세요.

    담당자의 메일이 변경된 경우

    2. "호환성 보기 설정"

    3. 리스트에 "katri.re.kr" 추가

    4. 인터넷 전부 끄고 다시 시도합니다. 

  • 담당자 이메일 변경/이메일 분실로 메일 확인이 불가합니다.

     

    기존 아이디/비밀번호 찾기는

    로그인 > 아이디 찾기 / 비밀번호 찾기 를 이용해 주시기 바랍니다.

    다만 담당직원 퇴사 및 이동, 회원가입 시 등록한 이메일 정보 분실 등의 사유로

    이메일 전송이 되지 않아 아이디 또는 패스워드 찾기가 어려운 경우 아래의 절차를 수행하여 주시기 바랍니다.

     

    1. 연구원 운영담당자에게 이메일 변경 메일 발송(고객)

    - 내용에 담당자 이메일 주소(계정) 기입

    - 사업자 등록증 사본첨부 (필수)

    - 운영담당자 이메일 주소: system@katri.re.kr

     

    2. 연구원 담당자가 처리 후 회신(연구원)

    - 회원가입 이메일 정보 변경

    - 아이디, 패스워드 찾기 방법 회신

     

    3. 메일 주소 변경 완료 회신 확인 후 ID/PW 찾기 진행(고객)

     

    4. 변경된 메일을 통해 회신받은 아이디/임시 비밀번호로 로그인(고객)

     

    감사합니다.
